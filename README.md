### 🚀 Main Projects

####　catrender
[https://github.com/manahoinagaki/catlendar]
**「共有カレンダー・家計簿・ToDo」を統合したライフログ・アプリ**（開発中）
Web版でのプロトタイプを経て、現在はモバイルアプリとしてのリリースを目指し、Dart (Flutter) でフルスクラッチ開発を行っています。

**Current Status**: 
    * カレンダー機能のコアロジックを実装中
    * 将来的に iOS / Android 両 OS でのリリースを目標としています

**Stack**: Dart (Flutter)

**Point**: 複雑なカレンダーUIの描画とデータ構造の最適化に挑戦しています。Web版（Next.js）での開発経験を活かし、よりネイティブアプリとして使い勝手の良いUI/UXを追求しています。

#### 家計簿アプリ（Next.js版）
[https://github.com/manahoinagaki/household_account_book]
catrenderの機能の核となる「家計簿」の仕組みのWebアプリケーションです。

Status: 大枠の機能は完成済み

Stack: Next.js, TypeScript

Focus: 収支データの可視化と、入力のしやすさを意識したUI設計。ここで得た知見を現在のモバイル版開発にフィードバックしています。

#### UnFes1-23 (Internal Project)
大学祭の企画運営をサポートするWebアプリケーションです。実際のイベントでの運用を想定し、利便性とリアルタイム性を重視して開発しました。

**Status**: 🛠️ Development (Currently Private for Security)

※個人情報保護の観点から現在は非公開としていますが、一般公開に向けてリファクタリングを実施中です。

**Key Features**:

Real-time Updates: Firebaseを用いた企画情報のリアルタイム配信

Dynamic Content: Next.js (App Router) による柔軟なコンテンツ管理

Stack: Next.js, TypeScript, Firebase (Firestore / Auth)

Point: 多くの来場者が同時にアクセスする状況を想定し、Firestoreのデータ設計を最適化しました。また、管理者・ユーザー双方にとって迷いのない、直感的なUI/UXの提供にこだわっています。
